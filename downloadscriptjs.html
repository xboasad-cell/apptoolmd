Bot Configuration Guide

Follow these steps to configure your Telegram bot and Geo API key for the script to work properly.

1. Telegram Bot Setup

You need to configure your Telegram bot by setting the bot token and chat ID:

  1. Open Telegram and search for @BotFather
  2. Create a new bot using the /newbot command
  3. Follow the instructions and save the API token provided
  4. To get your Chat ID, send a message to your bot
  5. Visit https://api.telegram.org/bot<YOUR_TOKEN>/getUpdates
  6. Find your chat ID in the response JSON
Bot Token: YOUR_BOT_TOKEN_HERE
Chat ID: YOUR_CHAT_ID_HERE

2. Geo API Key Setup

To get your Geo API key, follow these steps:

  1. Go to ipgeolocation.io/signup and create an account
  2. After registration, visit ipgeolocation.io/requestApiLogs
  3. Copy your API key from the dashboard
  4. Click the "Request Logs" button to activate your key
  5. Replace the placeholder in the script with your actual API key
geoApiKey: "YOUR_GEO_API_KEY_HERE"
Download Script
Important Notes:
  • Keep your API keys and tokens secure - never share them publicly
  • After downloading the script, open it in a text editor and replace the placeholder values with your actual keys
  • The script requires both the Telegram bot configuration and the Geo API key to function properly
  • Make sure to click "Request Logs" on the ipgeolocation website to activate your API key
